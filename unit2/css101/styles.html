BLOCK ELEMENT

  1. A Block element, by default, takes up all of the space to the right and left, internal to it's parent
  2. A Block can can have height and width
  3. Nothing can ever be on either side (internal to the parent)because it's 100%. if you change the width, it will just add margin. Blocks will ignore margin you add, because they already have 100%
  4. Can be > 100%

Inline elements

  1. 1. Respect left and right padding, left and right margin but will ignore top and bottom
  2. 2. Couldn't care less what is left or right.
  3. 3. With a few exceptions (namely images) you CANNOT tell an inline element how big to be.
  4. 4. Will happily ignore all block level rules unless forced to do so

  3. 3. Can have top/bottom padding/margin (block positive)
